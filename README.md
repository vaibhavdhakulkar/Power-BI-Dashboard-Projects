# 📊 Power BI Dashboard Projects

## 📌 Overview: <br>
This repository contains a collection of interactive dashboard projects developed using **Microsoft Power BI**.  
The purpose of these projects is to practice **data transformation, data modeling, and data visualization** while extracting meaningful business insights from real-world datasets.

These dashboards demonstrate how raw data can be converted into clear, interactive, and user-friendly reports for decision-making.

---

### 🎯 Objectives
- Perform data cleaning and transformation using Power Query  
- Analyze datasets using Power BI  
- Create interactive dashboards and reports  
- Visualize trends and patterns using charts and KPIs  
- Gain hands-on experience with Power BI and DAX  
- Build a strong Business Intelligence (BI) portfolio  

---

### 🛠️ Tools & Technologies Used
- Microsoft Power BI  
- Power Query  
- DAX (Data Analysis Expressions)  
- Excel / CSV datasets  
- Data Modeling & Relationships  

---

### 📂 Projects Included
Some of the projects in this repository include:
- 🏥 **Hospital Data Analysis Dashboard** – Patient, department, and revenue analysis  
- 🚕 **Uber Data Analysis Dashboard** – Trip trends and peak hour analysis  
- 🍔 **Zomato Sales Analysis Dashboard** – Order trends and food category insights  
- 🎵 **Spotify Data Analysis Dashboard** – Artist popularity and listening trends
  
*(More dashboards will be added regularly)*

---

### 🔍 Key Features:
- Interactive dashboards with slicers and filters  
- KPI cards for quick performance overview  
- Data cleaning and transformation using Power Query  
- DAX measures and calculated columns  
- Clear and user-friendly visual design  
- Business insights and conclusions from data
