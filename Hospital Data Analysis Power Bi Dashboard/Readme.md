# 🏥 Hospital Data Analysis Dashboard (Power BI)

This project focuses on analyzing hospital case data using Microsoft Power BI to provide meaningful insights into healthcare service distribution across districts and medical specialties. The dashboard is designed to help understand patient patterns, specialty-wise workload, and regional healthcare performance through interactive and visual analytics.<br>

The data model follows a star schema, where the Hospital Data table acts as the fact table and is connected to two dimension tables: District and Specialty Mapping. The District table enables district-wise analysis using district codes and names, while the Specialty Mapping table groups individual medical specialties into broader specialty categories, allowing higher-level analysis and comparison.<br>

The dashboard analyzes hospital cases based on multiple dimensions such as district, specialty group, patient type (adult/child), age profile, time bands, and archive date. Key performance indicators (KPIs) display total cases and patient distribution, while charts and graphs highlight district-wise case volume, specialty group performance, age-based trends, and time-band analysis.<br>

Interactive slicers and filters allow users to drill down into specific districts, specialties, patient categories, and time periods, making the dashboard dynamic and user-friendly. This project demonstrates practical skills in data modeling, data cleaning, DAX measures, and visualization design, and serves as a real-world example of healthcare analytics using Power BI.<br>
